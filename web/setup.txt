Setup:
	Install node.js from nodejs.org

	Install the Apple Developer Tools -> Command Line Tools for Xcode

	Install Redis
		wget http://redis.googlecode.com/files/redis-2.4.15.tar.gz
			or latest version...
		tar xzf redis-2.4.15.tar.gz
		cd redis-2.4.15
		sudo make install

	Install gevent
		sudo pip install gevent
		if doesn't work, should do:
			curl -O http://pypi.python.org/packages/source/g/gevent/gevent-0.13.6.tar.gz
				or latest version...
			tar xf gevent-0*
			cd gevent-0.13.6
			./fetch_libevent.py
			python setup.py build install

	Install ZeroMqls
		http://download.zeromq.org/zeromq-3.2.0-rc1.tar.gz
		tar xzf zeromq-3.2.0-rc1.tar.gz
		cd zeromq-3.2.0-rc1
		./configure
		make
		sudo make install

	Install ZeroRpc
		sudo pip install zerorpc

	Install IcedCoffeeScript by running "npm install -g iced-coffee-script"

	Install CoffeeScript syntax in Sublime
		cd ~/Library/Application\ Support/Sublime\ Text\ 2/Packages
		git clone git://github.com/dhoelzgen/iced-coffee-script-tmbundle.git IcedCoffeeScript

	Run "npm -d install" in the web/back/ directory.
